Software Engineering Intern (Flutter | Java Spring Boot | Angular)

Location: Pune / Hybrid (Office + Remote)
Duration: 6 Months (Internship)
Compensation: ₹10,500 per month (Indian Rupees)
Start Date: Immediate / As mutually agreed

About Rebind

Rebind is a technology startup building solutions in the consumer trust, product authentication, and brand engagement space. Our platform helps brands verify product authenticity using QR codes, connect directly with end consumers via WhatsApp/web, and gain real-time insights — helping fight counterfeiting and improve consumer confidence.

As a startup, we work on real problems with real customers. Interns at Rebind are not observers — they actively contribute to production systems and product decisions.

Role Overview

We are looking for a Software Engineering Intern who is passionate about technology and eager to learn by building real-world systems. You will work closely with the founding and engineering team to design, develop, and improve our web and mobile applications.

This role is ideal for someone who enjoys hands-on coding, takes ownership of tasks, and can work independently without constant supervision.

Technical Skill Requirements

The candidate should have working knowledge of one or more of the following (exposure to all is a strong plus):

  • Flutter – Mobile app development
  • Java + Spring Boot – Backend APIs and services
  • Angular – Frontend web applications
  • REST APIs, basic database concepts, and Git version control

Key Responsibilities

  • Develop and enhance backend APIs using Java Spring Boot
  • Build and maintain frontend features using Angular
  • Contribute to mobile app development using Flutter
  • Write clean, readable, and maintainable code
  • Debug issues and improve system performance
  • Collaborate with product and operations teams to understand requirements
  • Take ownership of assigned tasks and deliver within timelines

What You Will Learn (Startup Exposure)

  • End-to-end product development (frontend, backend, mobile)
  • How production-grade SaaS systems are designed and scaled
  • API design, authentication, and data flows
  • Working directly with founders and decision-makers
  • Understanding business problems and converting them into technical solutions
  • Real accountability — your work will go live and be used by customers

This is a high-learning, high-ownership internship, ideal for someone who wants to grow fast.

Ideal Candidate Profile

  • Passionate about software engineering and technology
  • Quick learner with a problem-solving mindset
  • Self-motivated and accountable for assigned work
  • Comfortable working independently without constant hand-holding
  • Strong logical thinking and debugging skills
  • Open to feedback and continuous improvement

Education:

Final-year students or recent graduates in Computer Science / IT / related fields. Strong self-taught developers are also welcome.
